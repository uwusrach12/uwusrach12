GIT flow

GIT clone- для того, что бы "склонить" репозиторий

GIT pull- для того, что бы получить изменени с репозитория

GIT checkout -b branchname- создать новую ветку и переключиться на нее

GIT add .- добавить изменения, которые сделали локально

GIT commit -m "commit text"- подписать их

GIT push origin brachname- запушить на git сервер
